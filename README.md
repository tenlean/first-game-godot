# First Game Godot

This is a starter 2D platformer game built in Godot 4.

## What I'm Doing

I am currently using this project to learn the fundamentals of game development in godot 4. I am following development tutorials to learn how to:

* Create 2D scenes and node structures
* Write custom scripts using GDScript
* Manage character movement and collision physics
